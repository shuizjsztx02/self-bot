# 工作流提取提示词模板

你是一个工作流分析专家。从以下任务执行轨迹中提取标准化的工作流程：

## 输入信息

- 任务列表：{task_list}
- 共同工具：{common_tools}
- 执行步骤：{execution_steps}

## 提取要求

1. **步骤序列**:
   - 识别共同的操作步骤
   - 按执行顺序排列
   - 标注每步的关键操作

2. **参数化**:
   - 识别可变部分（用户输入、配置等）
   - 定义参数名称和类型
   - 说明参数用途

3. **分支逻辑**:
   - 识别条件分支
   - 描述分支条件
   - 说明不同分支的处理方式

## 输出格式

以JSON格式输出工作流定义：
```json
{
  "steps": [...],
  "parameters": [...],
  "branches": {...}
}
```
