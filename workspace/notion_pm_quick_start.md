# Notion 项目管理系统快速入门

## 5分钟快速设置

### 第一步：创建项目数据库
1. 新建页面 → 选择 **Table** → 命名为 **"项目"**
2. 添加以下属性：
   - `状态` (Select): 规划中/进行中/已完成
   - `负责人` (Person)
   - `截止日期` (Date)
   - `进度` (Number)

### 第二步：创建任务数据库
1. 在项目页面内新建数据库 → **Table** → 命名为 **"任务"**
2. 添加以下属性：
   - `所属项目` (Relation) → 关联到"项目"数据库
   - `状态` (Select): 待办/进行中/已完成
   - `优先级` (Select): P1/P2/P3
   - `负责人` (Person)
   - `截止日期` (Date)

### 第三步：创建查询视图

#### 视图1：项目看板
- 在项目数据库点击 **+ Add a view**
- 选择 **Board** → 命名为 **"项目状态"**
- 分组依据：`状态`

#### 视图2：任务看板
- 在任务数据库点击 **+ Add a view**
- 选择 **Board** → 命名为 **"任务状态"**
- 分组依据：`状态`

#### 视图3：我的任务
- 添加新视图 **Table** → 命名为 **"我的任务"**
- 添加筛选器：`负责人` 是 `我`

#### 视图4：本周任务
- 添加新视图 **Calendar** → 命名为 **"任务日历"**
- 日期属性：`截止日期`
- 添加筛选器：`截止日期` 在 `本周内`

### 第四步：创建仪表板
1. 新建页面 → 命名为 **"项目仪表板"**
2. 添加以下数据库视图：
   ```
   /view 项目.项目状态
   /view 任务.任务状态
   /view 任务.我的任务
   /view 任务.任务日历
   ```

### 第五步：创建新任务（快速方法）
1. 在任务数据库中点击 **New**
2. 填写任务信息：
   - 任务名称：输入任务标题
   - 所属项目：选择关联的项目
   - 状态：选择状态
   - 优先级：选择优先级
   - 负责人：@自己或团队成员
   - 截止日期：选择日期

## 常用查询示例

### 1. 查询所有进行中的项目
```
filter(prop("状态") == "进行中")
```

### 2. 查询高优先级任务
```
filter(prop("优先级") == "P1")
```

### 3. 查询今天到期的任务
```
filter(prop("截止日期") == today())
```

### 4. 查询我的未完成任务
```
filter(
  and(
    prop("负责人") == currentUser(),
    prop("状态") != "已完成"
  )
)
```
